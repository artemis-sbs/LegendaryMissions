# Legendary Missions

Legendary Missions recreates the feel of the classic out-of-the-box Artemis
missions for **Artemis: Cosmos** &mdash; and provides a library of reusable
**addons** you can build your own missions on.

[How to play](playing/index.md){ .md-button }
[Hosting](hosting/index.md){ .md-button }
[Addon reference](addons/index.md){ .md-button }

---

Writing your own mission? The MAST language, GUI/comms/science/AI recipes, and the
full API live in the
[sbs_utils documentation](https://artemis-sbs.github.io/sbs_utils/). This site
covers **playing** and **hosting** Legendary Missions, and the **addons** it adds
on top of sbs_utils.
